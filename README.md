# 简介
这款应用是音频创作的极致艺术之选，旨在为您提供一站式解决方案，轻松应对音频剪辑、播客源生成和音乐制作等各种挑战。
其丰富而强大的功能集合，不仅能够让您轻松编辑音频内容，还能够打造出引人入胜的播客节目和令人心旷神怡的音乐作品，作为苹果播客应用的理想伴侣。

本应用的前身是集成于 Movie-bot 一款插件，详细请见 [说明文档](https://alano-i.github.io/doc/audio_tools/)
